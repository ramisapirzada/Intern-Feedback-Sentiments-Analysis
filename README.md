# Intern-Feedback-Sentiments-Analysis
Machine Learning project for analyzing intern feedback using sentiment analysis.
Intern Feedback Sentiment Analysis

## Project Overview

This project analyzes internship feedback and classifies sentiments as Positive or Negative using Machine Learning.

## Objective

To identify intern satisfaction levels and highlight areas for improvement in internship programs.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Methodology

1. Dataset Creation
2. Data Preprocessing
3. TF-IDF Vectorization
4. Train-Test Split
5. Logistic Regression Training
6. Model Evaluation

## Results

The Logistic Regression model achieved 100% accuracy on the generated test dataset.

## Future Enhancement

Transformer-based models such as BERT can be used to improve contextual understanding and sentiment classification performance.
